## Organizador de Downloads

Uma ferramenta de linha de comando que automatiza a organização da pasta de Downloads (ou qualquer outra pasta que contenha arquivos variados), feita para explorar, com intuitos de estudos, a biblioteca **System.CommandLine** na construção de apps console com CLI do .NET.

## > Funcionalidades

- Interface CLI: construída com a biblioteca NuGet System.CommandLine (v2.0.0-beta4.22272.1) do .NET
- Organização Automática: move arquivos para pastas categorizadas automaticamente
- Múltiplas Estratégias: organiza por extensão ou data de modificação
- Estatísticas: Retorna com infos sobre quantos arquivos foram movidos, etc

## > Tecnologias aplicadas

- .NET 8
- System.CommandLine
- Stratety Pattern
- Onion Architecture
